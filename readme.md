# Gerenciador de Tarefas

Projeto desenvolvido para praticar conhecimentos adquiridos em Python, utilizando funções, classes e persistência de dados em banco.

## Visão Geral

Este projeto tem como objetivo criar um gerenciador de tarefas focado na aplicação de conceitos de desenvolvimento backend em Python, boas práticas de código e organização da lógica da aplicação.

## Principais Funcionalidades

- Adicionar tarefas
- Listar tarefas
- Remover tarefas

## Tecnologias Utilizadas

- Python — linguagem principal do projeto.
- SQLite3 — banco de dados utilizado para persistência das informações.
